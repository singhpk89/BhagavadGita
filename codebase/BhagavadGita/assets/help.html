Shrimad Bhagvad Gita Features
------------------------

To Open Menu Drawer : Launch App --> Click on Menu drawer (top left icon OR title at Header) OR perform Left to Right Swipe from left most corner

To See Chapters : Open Menu Drawer --> Select Chapters from Drawer Items

To See Quotes Of A Chapter : Open Menu Drawer --> Select Chapters from Drawer Items --> Select any particular Chapter

To See Quote Detail : Open Menu Drawer --> Select Chapters from Drawer Items --> Click any particular Chapter > Click any particular Quote

Go To Quote : Open Quote Detail using above step --> Select particular Chapter and Quote using Respective Spinner at top

How To See Next Quote : Open Quote Detail --> Swipe Towards Left

How To See Previous Quote : Open Quote Detail --> Swipe Towards Right

Add To/Remove From Favorite : Open Quote Detail Screen --> Click On Favorite Icon (at header)

To Share Quote : Open Quote Detail Screen --> Click On Share Icon (at header)

How To Launch Settings : Open Menu Drawer --> Select Settings from Drawer Items

Set Font Size : Launch Settings Screen --> Increase/Decrease Font Size

Set Show/Hide Shloka : Launch Settings Screen --> Disable 'Show Shloka'

To Enable/Disable Auto Talk of Verse : Launch Settings Screen --> Enable/Disable 'AutoTalk'

Set Un/Read Shloka From Verse : Launch Settings Screen --> Enable/Disable 'Read Shloka'
